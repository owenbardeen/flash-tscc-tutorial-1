FLASH on TSCC Tutorial

A step-by-step guide for building and running FLASH 4.7.1 on SDSC's TSCC cluster.

Prerequisites

  • TSCC account with access to the cluster
  • Basic knowledge of Linux command line
  • Familiarity with HPC concepts
  • Git installed on your local machine

Getting Started

This tutorial is organized in a logical progression:

  1. Installation Guide - Set up FLASH on TSCC
  2. First Simulation - Run a basic Sedov test
  3. Advanced Topics - Optimize and customize
  4. Troubleshooting - Common issues and solutions
Note: All commands in this tutorial assume you're working on TSCC login nodes.

Installation Guide

1. Log into TSCC

ssh your_username@tscc.sdsc.edu

2. Load Required Modules

module load intel
module load mvapich2_ib
module load hdf5
module load python

3. Download FLASH 4.7.1

cd /home/your_username
wget https://flash.rochester.edu/site/flashcode/user_support/flash4.7.1.tar.gz
tar -xzf flash4.7.1.tar.gz
cd flash4.7.1

4. Configure and Build

# List available setups
./setup -list

# Configure Sedov test problem
./setup Sedov -auto

# Build FLASH
cd object
make -j4

5. Verify Installation

# Check if flash4 executable exists
ls -la flash4

# Check version
./flash4 -v
Tip: The build process may take 10-30 minutes depending on system load.

Common Build Options

# With parallel I/O support
./setup Sedov -auto +parallelIO

# With UG (unsplit grid) solver
./setup Sedov -auto +ug

# 3D version
./setup Sedov -3d -auto

First Simulation: Sedov Test

1. Prepare Simulation Directory

# Create a run directory
cd /home/your_username
mkdir flash_runs
cd flash_runs
mkdir sedov_test
cd sedov_test

# Copy necessary files from installation
cp ../../flash4.7.1/object/flash4 .
cp ../../flash4.7.1/source/Simulation/SimulationMain/Sedov/Config .
cp ../../flash4.7.1/source/Simulation/SimulationMain/Sedov/Flash.par .

2. Create PBS Submission Script

Create a file called submit.pbs:

#!/bin/bash
#PBS -q normal
#PBS -l nodes=1:ppn=16
#PBS -l walltime=01:00:00
#PBS -N flash_sedov
#PBS -o flash.out
#PBS -e flash.err

cd $PBS_O_WORKDIR

# Load modules
module load intel mvapich2_ib hdf5

# Run FLASH
mpirun -np 16 ./flash4

3. Submit the Job

# Make script executable
chmod +x submit.pbs

# Submit to queue
qsub submit.pbs

# Check job status
qstat -u $USER

4. Monitor Simulation Progress

# Check output files
ls -la *.log

# Watch log file (Ctrl+C to stop)
tail -f flash_sedov.log

# Check for checkpoint files
ls -la chk* 2>/dev/null || echo "No checkpoints yet"

5. Analyze Results

# Convert HDF5 output to VTK for visualization
module load visit

# Or use Python with h5py
module load python
python3 -c "import h5py; import numpy as np; print('HDF5 available')"

Advanced Topics

Performance Tuning

Compiler Optimization

# Edit setup script for optimization
cd flash4.7.1
cp sites/TSCC/Makefile.h sites/TSCC/Makefile.h.backup

# Add optimization flags
echo "FFLAGS_OPT = -O3 -xHost -ipo" >> sites/TSCC/Makefile.h
echo "CFLAGS_OPT = -O3 -xHost -ipo" >> sites/TSCC/Makefile.h

MPI Configuration

# Test different MPI configurations
mpirun -np 32 -ppn 16 ./flash4  # 2 nodes, 16 cores each
mpirun -np 64 -ppn 8  ./flash4  # 8 nodes, 8 cores each

Custom Physics Modules

# Create custom setup
./setup -n MyCustomSetup
cd source/Simulation/SimulationMain/MyCustomSetup

# Edit Config file to include desired physics
# Example: add gravity + multiphysics
+gravity
+multiphysics

Parameter Studies

#!/bin/bash
# Script for parameter sweep
for density in 1.0 2.0 5.0 10.0
do
    sed -i "s/^basenumdens.*/basenumdens = $density/" Flash.par
    mpirun -np 16 ./flash4
    mv flash.log flash_density_${density}.log
done

Parallel I/O Optimization

# Configure with parallel HDF5
./setup Sedov -auto +parallelIO

# In Flash.par, adjust I/O settings
restart = .false.
plotFileNumberStep = 10
checkpointFileNumberStep = 50
ioStrategy = "collective"

Troubleshooting

Build Issues

Error: Missing libraries

# Solution: Verify all modules are loaded
module list

# Load required modules
module purge
module load intel mvapich2_ib hdf5 python

Error: "make: command not found"

# Solution: Load GNU compilers
module load gnu
make clean
make

Error: Permission denied on /home

# Solution: Check disk quota
quota -s

# Clean up old files
cd /home/your_username
du -sh ./* | sort -h

Runtime Issues

Error: MPI initialization failed

# Solution: Check MPI environment
which mpirun
mpirun --version

# Test MPI with simple program
mpirun -np 4 hostname

Error: HDF5 library not found

# Solution: Load HDF5 module
module load hdf5/1.10.6

# Or specify library path
export LD_LIBRARY_PATH=/opt/hdf5/lib:$LD_LIBRARY_PATH

Job stuck in queue

# Check queue status
qstat -Q

# Try different queue
#PBS -q gpu-shared  # or normal, gpu, etc.

Common TSCC Issues

Session timeout

# Use screen or tmux to persist sessions
module load screen
screen -S flash_session

# Or use nohup for long commands
nohup make -j8 > build.log 2>&1 &

Memory issues

# Request more memory in PBS script
#PBS -l pmem=4gb
#PBS -l mem=64gb
